# MENU
Creacion de un script que muestra un menú con varias opciones a eligir controlando cualquier boton que se pulse

# EnviarCorreo
Creacion de un script que mandandole una direccion de correo, un asunto y un texto, envia un correo a la direccion proporcionada

# ActualizarSistema
Creacion de script que actualiza y limpia paquetes del sistema mediante ciertos comandos

# CifrarArchivo
Creacion de script que te da a eligir entre los archivos del sistema, pudiendo elegir cual cifrar con contraseña y despues creandote un enlace duro para poder acceder a dicho fichero
